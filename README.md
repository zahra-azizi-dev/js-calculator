# js-calculator
# 🧮 Modern Calculator App

A modern and responsive calculator built with **HTML**, **JavaScript**, **Tailwind CSS**, and a touch of **Pure CSS** for custom styling and animations.

📸 Preview
(https://zahra-azizi-dev.github.io/js-calculator/)

---
 ✨ Features

✅ Responsive Design

✅ Basic Arithmetic Operations

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (÷)

✅ Operator Validation

* Prevents duplicate operators
* Handles negative numbers correctly

✅ Decimal Number Support

✅ Backspace Functionality

✅ Clear Screen Functionality

✅ Operation Priority

* Multiplication and Division before Addition and Subtraction

✅ Error Handling

* Division by zero protection

---
 🌙 Dark / Light Mode

A custom Dark Mode & Light Mode switch with animated SVG icons:

☀️ Sun Animation

🌙 Moon Animation

✨ Smooth transitions and visual effects

Theme preference is automatically saved using **Local Storage**, so your selected mode remains active even after refreshing the page.
